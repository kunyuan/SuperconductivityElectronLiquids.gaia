# #06 $\mu^*$ 作为唯象参数

> 由于缺乏可靠的微观计算，Coulomb 赝势 $\mu^*$（描述低能配对通道中有效 Coulomb 排斥强度的无量纲参数）通常被作为可调参数处理，经验值在 0.1--0.2 范围内。

## 背景

$\mu^*$ 的经验取值范围 0.1--0.2 最初由 Morel 和 Anderson (1962) 建议，基于静态屏蔽近似下的粗略估计。此后的实践中，$\mu^*$ 常被调整以使理论 $T_c$ 与实验吻合，而非从微观理论推导。某些过渡金属化合物（如 V、Nb$_3$Sn）甚至需要 $\mu^* = 0.2$--$0.5$，远超原始估计。本文通过 vDiagMC 计算消除了这一唯象性。
