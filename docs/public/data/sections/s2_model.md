# The Model and Basic Relations

The theoretical foundation begins with an exact decomposition of the electron-phonon action into the complete many-electron action $S_e$, the phonon action $S_{\mathrm{ph}}$ with physical dispersion, the electron-phonon coupling $S_{e\text{-ph}}$, and a counterterm $S_{\mathrm{CT}}$ that prevents double-counting of the static polarization already built into the phonon frequencies. Under the adiabatic condition $\omega_D / E_F \ll 1$ (belief 0.90, prior 0.95), Migdal's theorem guarantees that phonon vertex corrections are suppressed at $O(\omega_D / E_F)$, establishing the Migdal-Eliashberg formalism as a controlled low-energy theory (belief 0.95). This energy-scale separation allows the kernel of the Bethe-Salpeter equation to be cleanly decomposed into a purely electronic four-point vertex $\tilde{\Gamma}^e$ encoding all non-perturbative Coulomb effects and a phonon-mediated interaction $W^{\mathrm{ph}}$ (belief 1.00 for the kernel decomposition). The precursory Cooper flow provides the bridge to observable physics: the anomalous vertex on the Fermi surface obeys $\Lambda_0 = 1/(1 + g \ln(\omega_\Lambda / T))$, which diverges at $T_c = \omega_\Lambda e^{1/g}$ when the dimensionless coupling $g$ is negative (belief 0.90). By computing $g$ in the normal state, one can extract $T_c$ without solving the full superconducting state --- a crucial simplification that underpins all the material predictions in later sections.
