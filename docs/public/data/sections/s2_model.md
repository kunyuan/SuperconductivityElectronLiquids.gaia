# The Model and Basic Relations

This section constructs the formal field-theoretic foundation on which the entire downfolding program rests. The electron-phonon action is decomposed into an exact many-electron part $S_e$, a phonon part $S_{\mathrm{ph}}$, a coupling term $S_{e\text{-ph}}$, and a counterterm $S_{\mathrm{CT}}$ that prevents double-counting of static screening already present in the physical phonon dispersion (belief 0.95). From the Migdal-Eliashberg framework established in Section I, Migdal's theorem guarantees that phonon vertex corrections to the Bethe-Salpeter equation kernel are suppressed at $O(\omega_D/E_F)$, allowing the BSE kernel to be cleanly decomposed into a purely electronic four-point vertex $\tilde\Gamma^e$ and the phonon-mediated interaction $W^{\mathrm{ph}}$ (belief 0.998). This decomposition is the structural prerequisite for the downfolding in Section III. The section also introduces the precursory Cooper flow (PCF) formalism (belief 0.90), which establishes that the anomalous vertex diverges as $\Lambda_0 \sim 1/(1 + g\ln(\omega_\Lambda/T))$ near the Cooper instability, providing a practical method to extract $T_c$ from normal-state calculations by extrapolating the divergence temperature $T_c = \omega_\Lambda e^{1/g}$.
