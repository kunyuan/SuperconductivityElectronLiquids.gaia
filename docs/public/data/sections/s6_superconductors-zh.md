# 简单金属的从头算预测

所有理论线索汇聚为一个无参数工作流：(1) 通过 vDiagMC 从均匀电子气四点顶角计算 $\mu_{E_F}$，(2) 映射到材料的 $r_s$ 并通过 BTS 关系缩放至 $\mu^*$，(3) 从 DFPT 获取 $\lambda$，(4) 求解降阶 Eliashberg 方程。该复合结论达到置信度 0.99（先验 0.50），互信息 0.10 比特，依赖四个前提：降阶 BSE (0.76)、vDiagMC 的 $\mu$ 值 (0.50)、DFPT 可靠性 (0.86) 和均匀电子气参数化映射 (0.83)。对于铝（$r_s = 2.07$），工作流给出 $\mu^* \approx 0.11$ 和 $T_c = 1.1 \pm 0.3$ K，与实验值 1.2 K 吻合良好——预测达到置信度 0.93，互信息 0.67 比特，结合工作流与实验锚点，优于高估 58% 的唯象预测。锌（$r_s = 2.31$）呈现相同模式：$T_c^{\mathrm{th}} = 0.7 \pm 0.3$ K 对比 $T_c^{\mathrm{exp}} = 0.875$ K，同样为置信度 0.93。最引人注目的成功是锂（$r_s = 3.25$），其中 $\mu^* \approx 0.16$ 几乎完全抵消了 $\lambda \approx 0.41$，正确预测了亚毫开尔文的 $T_c$，置信度 0.96（互信息 0.60 比特）——唯象方法偏差达三个数量级。在已校准的金属之外，钠（$r_s = 3.93$）被预测其大的 $\mu^*$ 超过弱电子-声子耦合，导致无 Cooper 不稳定性；镁（$r_s = 2.66$）处于亚纳开尔文区间。两者均靠近超导与非超导基态之间的量子相变，在该处 $T_c = \omega_\Lambda e^{1/g}$ 随耦合 $g \to 0$ 呈指数灵敏（置信度 0.79，互信息 0.60 比特）。在静水压力下，铝的 $T_c$ 被预测呈非单调变化，先升后降，与实验压力研究一致（置信度 0.79，互信息 0.60 比特）。
