# 常规超导体

本节组装了完整的从头算 $T_c$ 预测工作流，并将其与五种简单金属的实验结果进行对比。工作流（置信度 0.99）通过复合策略结合三个组成部分——降阶 BSE、vDiagMC 导出的 $\mu^*$、以及 DFPT 计算的 $\lambda$——首先建立简单金属中 $\mu^*$ 的可获得性（尽管由于 vDiagMC 不确定性的传播，这一中间声明的置信度仅为 0.40），然后演绎得出完整的无参数管线。对于铝（$T_c^{\mathrm{th}} = 1.1 \pm 0.3$ K，实验值 1.2 K，置信度 0.93）和锌（$T_c^{\mathrm{th}} = 0.7 \pm 0.3$ K，实验值 0.875 K，置信度 0.93），与实验数据的溯因推理强烈支持从头算预测，而非唯象替代方案（二者均高估约 57%）。最引人注目的成功是锂：从头算预测 $T_c \lesssim 10^{-3}$ K（置信度 0.96）与实验值（$4 \times 10^{-4}$ K）一致，而唯象预测偏高三个数量级，置信度降至 0.13。该框架还预测了铝 $T_c$ 的非单调压力依赖性（置信度 0.79），并将钠和镁定位在超导-非超导量子相变 (quantum phase transition) 边界附近（置信度 0.79），在该处 $\mu^*$ 几乎或完全抵消 $\lambda$，使超导性呈指数级脆弱。
