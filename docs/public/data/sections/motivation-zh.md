# 动机与背景

传统超导理论通过 McMillan 或 Allen-Dynes 公式预测转变温度 $T_c$，需要两个输入：由 DFPT 计算的电子-声子耦合常数 $\lambda$，以及描述 Cooper 配对通道中有效电子-电子排斥的库仑赝势 $\mu^*$。由于 $\mu^*$ 无法从第一性原理可靠计算，通常被当作 0.1--0.2 范围内的自由参数。这种唯象自由度对于强耦合超导体尚可接受，但对 $T_c$ 在开尔文或亚开尔文量级的金属而言，指数灵敏性 $T_c \propto \exp(-1/g)$ 会将 $\mu^*$ 的微小不确定性放大为 $T_c$ 的数量级误差。唯象预测清楚地展示了这一问题：对于锂，取 $\mu^* = 0.1$ 得到 $T_c \approx 0.35$ K，而实验值为 $4 \times 10^{-4}$ K，高估了三个数量级（置信度 0.13，先验 0.10）。对于铝，预测的 1.9 K 超出实测 1.2 K 达 58%（置信度 0.40，先验 0.35）；对于锌，预测的 1.37 K 超出实测 0.875 K 达 57%（置信度 0.40，先验 0.35）。实验 $T_c$ 值本身则非常确定，三种金属的置信度均为 1.00。另一方面，RPA 对动态屏蔽库仑相互作用的处理在 $r_s \gtrsim 2$ 时预测出吸引性的 $\mu^* < 0$，但与受控的 vDiagMC 结果对比后，该声明的置信度从先验 0.50 降至 0.25，证实了 RPA 忽略顶角修正在金属密度区间产生了定性错误的结果。
