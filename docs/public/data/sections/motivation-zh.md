# 引言：动机与背景

本节提出了论文的核心问题：库仑赝势 (Coulomb pseudopotential, $\mu^*$) 能否从第一性原理计算得到，从而对简单金属的超导转变温度 (superconducting transition temperature, $T_c$) 做出定量预测？推理从确立 BCS 超导理论和绝热近似 ($\omega_D/E_F \ll 1$, 置信度 0.90) 出发，由此导出 Migdal-Eliashberg 框架（置信度 0.95）作为受控的低能有效理论。随后，本节系统梳理了唯象方法的失败：使用 McMillan 公式取 $\mu^* = 0.1$ 时，对铝的 $T_c$ 高估 58%，对锌高估 57%，对锂则高估了三个数量级。这些偏差驱动了对 $\mu^*$ 进行微观处理的需求。本节还引入了 BTS 重整化关系（置信度 0.99），该关系连接不同能量标度上的 $\mu^*$，并指出随机相位近似 (RPA) 在 $r_s \gtrsim 2$ 时预测出非物理的吸引性 $\mu^*$——该预测将在第四节中被 vDiagMC 的结果所矛盾。置信传播确认实验 $T_c$ 值仍然高度可信（铝和锌的置信度 $\geq 0.99$，锂为 1.00），而唯象预测则被大幅降权（铝和锌的置信度为 0.40，锂为 0.13），反映了其较差的解释力。
