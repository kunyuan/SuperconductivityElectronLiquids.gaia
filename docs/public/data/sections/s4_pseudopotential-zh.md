# 基于 vDiagMC 的库仑赝势计算

计算均匀电子气在金属密度区间（$r_s \in [1, 6]$）的粒子-粒子不可约四点顶角 $\tilde{\Gamma}^e$ 是一个长期挑战：裸微扰论在 $r_s \gtrsim 1$ 时发散，而 RPA 等部分重求和遗漏了关键的顶角修正。变分图形蒙特卡罗（置信度 0.83，先验 0.90）通过随机抽样粗线（自洽）传播子的费曼图来解决这一问题，避免了红外发散并能以受控误差棒外推至无穷阶。同伦展开（置信度 0.79，先验 0.88）通过连续变形裸库仑相互作用在每一微扰阶引入部分屏蔽，显著加速了收敛，在适中的图形阶数（$n \lesssim 7$）即可达到收敛。两者共同产生了整个推理图中互信息最高的边——0.74 比特——通向 $\mu_{E_F}(r_s)$ 的数值结果。然而结论本身停滞在置信度 0.50（先验 0.50）——噪声与推断模型要求两个前提同时成立，而它们各自的置信度（0.83 和 0.79）在严格合取下无法克服先验。计算得到的值为正且随 $r_s$ 单调递增：$r_s = 2$（铝类）时 $\mu_{E_F} \approx 0.21$，$r_s = 3.3$（锂类）时 $\mu_{E_F} \approx 0.33$。这些结果直接矛盾于 RPA 预测的吸引性 $\mu^*$，将矛盾节点的置信度推至 1.00，确认了 RPA 的预测是不受控近似的产物。
