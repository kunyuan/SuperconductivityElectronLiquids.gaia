# Bethe-Salpeter 方程的降阶

完整的动量-频率 Bethe-Salpeter 方程通过受控的降阶程序被约化为松原频率上的一维积分方程。配对传播子 $G_k G_{-k}$ 被精确分解为携带 Cooper 对数的低能相干部分 $\Pi_{\mathrm{BCS}}$ 和高能非相干余项。对相干部分进行动量求和后，角度依赖被吸收入态密度，留下纯频率核 $K(\omega, \omega') = \lambda(\omega, \omega') - \mu_{\omega_c}(\omega, \omega')$。关键假设是混合库仑和声子通道的交叉项在 $O(\omega_c^2 / \omega_p^2)$ 量级被压制（置信度 0.69，先验 0.90）——这是所有前提中置信度下降最大的，反映了中间截断能标层级的材料依赖性不确定性。结合绝热近似（置信度 0.90），推断将降阶 BSE 的结论从无信息先验 0.50 提升至置信度 0.76，互信息为 0.01 比特。铝参数下的玩具模型验证了定量准确性：完整 BSE 给出 $T_c/T_F = 10^{-5.668}$，降阶版本给出 $10^{-5.667}$，偏差仅 0.2%。降阶过程还为 $\lambda$（$W^{\mathrm{ph}}$ 的费米面平均，以准粒子因子加权，置信度 0.81）和 $\mu_{\omega_c}$（电子四点顶角在费米面上的投影，置信度 0.84）提供了微观定义。作为推论，BTS 重整化关系作为截断不变性的精确结果（置信度 0.99）而非唯象假设被导出，Morel-Anderson 常数赝势近似也由能量标度层级 $\omega_c \ll E_F$ 得到了微观证明（置信度 0.92）。
