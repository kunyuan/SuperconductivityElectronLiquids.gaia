# Bethe-Salpeter 方程的降阶

本节完成了核心理论推导：将完整的动量-频率 BSE 约化为可处理的纯频率积分方程。配对传播子被精确分解为相干的 BCS 型部分和非相干余项，其中混合库仑和声子通道的交叉项被证明在 $O(\omega_c^2/\omega_p^2)$ 量级被压制——不过这一压制声明（置信度 0.69）是推导链中最薄弱的结构环节。由 BSE 核分解和交叉项压制通过演绎推理得到降阶 BSE（置信度 0.76），即一个具有微观定义的 $\lambda$ 和 $\mu_{\omega_c}$ 核的一维积分方程。该框架通过玩具模型比较得到验证：完整 BSE 给出 $T_c/T_F = 10^{-5.668}$，而降阶版本给出 $10^{-5.667}$，二者仅差 0.2%（置信度约 0.76--1.00），通过溯因推理有力地支持了降阶的有效性。从降阶 BSE 出发，$\lambda$（置信度 0.81）和 $\mu$（置信度 0.84）均获得了微观定义，BTS 重整化关系也作为精确推论（$\mu$ 标度无关性，置信度 0.99）而非临时假设被导出——与历史上已知的 Bogoliubov-Tolmachev-Shirkov 关系建立了等价性。Morel-Anderson 常数赝势近似也从能量标度层级 $\omega_c \ll E_F$ 得到了微观证明（置信度 0.92）。
