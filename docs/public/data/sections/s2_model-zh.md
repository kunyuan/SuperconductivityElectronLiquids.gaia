# 模型与基本关系

理论基础从电子-声子作用量的精确分解出发：完整的多电子作用量 $S_e$、具有物理色散关系的声子作用量 $S_{\mathrm{ph}}$、电子-声子耦合项 $S_{e\text{-ph}}$，以及防止与物理声子频率中已包含的静态极化产生重复计数的抵消项 $S_{\mathrm{CT}}$。在绝热条件 $\omega_D / E_F \ll 1$（置信度 0.90，先验 0.95）下，Migdal 定理保证声子顶角修正在 $O(\omega_D / E_F)$ 量级被压制，确立了 Migdal-Eliashberg 形式体系作为受控低能理论的地位（置信度 0.95）。这一能量标度分离使得 Bethe-Salpeter 方程的核可以被清晰地分解为编码所有非微扰库仑效应的纯电子四点顶角 $\tilde{\Gamma}^e$ 和声子介导的相互作用 $W^{\mathrm{ph}}$（核分解的置信度为 1.00）。前驱 Cooper 流提供了通向可观测物理量的桥梁：费米面上的反常顶角满足 $\Lambda_0 = 1/(1 + g \ln(\omega_\Lambda / T))$，当无量纲耦合常数 $g$ 为负时在 $T_c = \omega_\Lambda e^{1/g}$ 处发散（置信度 0.90）。通过在正常态计算 $g$ 并外推，可以无需求解完整超导态即可提取 $T_c$——这一关键简化支撑了后续章节中所有材料的预测。
