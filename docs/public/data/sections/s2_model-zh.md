# 模型与基本关系

本节构建了整个降阶程序所依赖的场论基础。电子-声子作用量被分解为精确的多电子部分 $S_e$、声子部分 $S_{\mathrm{ph}}$、耦合项 $S_{e\text{-ph}}$，以及一个抵消项 $S_{\mathrm{CT}}$，用于防止与物理声子色散中已包含的静态屏蔽产生重复计数（置信度 0.95）。基于第一节建立的 Migdal-Eliashberg 框架，Migdal 定理保证了 Bethe-Salpeter 方程 (BSE) 核中的声子顶角修正在 $O(\omega_D/E_F)$ 量级被压制，从而允许 BSE 核被清晰地分解为纯电子四点顶角 $\tilde\Gamma^e$ 和声子介导的电子-电子有效相互作用 $W^{\mathrm{ph}}$（置信度 0.998）。这一分解是第三节降阶推导的结构前提。本节还引入了前驱 Cooper 流 (precursory Cooper flow, PCF) 形式体系（置信度 0.90），该体系表明反常顶角在 Cooper 不稳定性附近以 $\Lambda_0 \sim 1/(1 + g\ln(\omega_\Lambda/T))$ 发散，为从正常态计算外推发散温度 $T_c = \omega_\Lambda e^{1/g}$ 以获取 $T_c$ 提供了实用方法。
